# Counters

## Pages

### `/`

- Button to create a new counter starting at 0
- List of counters with their current value and a link to their page

### `/counter/:id`

- Heading with the counter's current value
- Button to increment the counter

### `/total`

- Heading with the total of all counters
- Button to reset all counters to 0

## Navbar

- Heading with link to the home page
- Link to the total page

## Guidelines

- Use `react-router-dom` and the React Context API
- Use fully controlled form components
- Use fully typesafe patterns
- Avoid large components and long code files
